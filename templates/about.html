Este é um experimento de site da Web construído com Python usando o framework minimalista Flask e o banco de dados SQLite.

Este aplicativo foi desenvolvido de forma colaborativa com o GitHub Copilot no Visual Studio Code.

Para obter o código fonte, instruções e para colaborar com o projeto, acesse o repositório oficial no GitHub.

Caso precise acessar o banco de dados diretamente, sugiro a ferramenta DB Browser que é leve e fácil de usar.

Para experimentos de hospedagem de forma gratuita você pode usar o Pythonanywhere que, inclusive, dá suporte a bancos de dados MySQL de forma limitada.

Sorte e sucesso!
